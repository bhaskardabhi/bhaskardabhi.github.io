Apache Error Notifier using Gulp

Gulp script which display notification if fatal error is logged in apache error log. As a lazy programmer, i created this script so i dont have manually search fatal errors in apache log.

How to use?

Install Npm. (If you don't know then please google.)

Install Gulp. npm install -g gulp

Install following depedencies by Gulp:

npm install --save gulp-notify gulp-intercept

Run: gulp in the directory of gulpfile.js

It will automatically display notification if fatal error is logged.

Note: By default path for apache error log file is /var/log/apache2/error.log. If yours is not same then add goto gulpfile.js and change logPath to path of your log file.
