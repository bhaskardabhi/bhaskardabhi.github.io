Siddu.js

Siddhu Paji, The great indian cricketer cum commentator. Siddhu.js is a script which display Siddhu Paaji's "Awesome" Quotes at different time interval. If you like the script then thoko Taali.

How to use?

Just call it by siddhu()

Following Configurations are allowed

{
  quotes: [],
  duration: null
}

qoutes: Script also allow others siddhu quotes to be passed as array

duration: Set custom time when quote should be displayed(In ms). By default it will be displayed at random time interval like it happen in his commentry.

Pass it as siddhu({quotes: ["Thoko Taali",'Thoko Another Taali']}, 600)
